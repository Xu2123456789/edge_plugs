# 📺 B站页面锁定器

> 🔒 专注B站视频内容，锁定导航页、分区页等干扰页面，只放行视频、搜索、个人空间、动态、收藏夹

## ✨ 功能特点

- **精准锁定**：只放行视频播放页、搜索页、个人空间、动态和收藏夹(存在问题)
- **一键开关**：工具栏图标快速启用/禁用
- **无干扰体验**：锁定页面无法刷新，防止分心
- **轻量级**：无后台服务，不影响浏览器性能

## 📦 安装方式

### 手动安装（开发者模式）
1. 开启「开发者模式」
2. 点击「加载已解压的扩展程序」
3. 选择项目文件夹

## ⚙️ 使用说明

- **启用锁定**：点击工具栏图标 → 开关变为蓝色
- **临时禁用**：点击工具栏图标 → 开关变为灰色
- **完全卸载**：右键工具栏图标 → 从Chrome中移除

## 🔒 隐私说明

本扩展**不收集任何用户数据**：
- 无网络请求
- 无数据存储
- 无远程服务
- 所有逻辑在本地运行
